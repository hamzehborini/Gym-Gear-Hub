Build a complete, production-quality, **frontend-only** website for a premium gym supplements brand called **"APEX FUEL"** (render the logo as a bold athletic wordmark).

## 1. Tech Stack — Strict Rules

Use ONLY:

- React 18 + Vite + TypeScript
- Tailwind CSS
- shadcn/ui components
- Lucide React icons
- Framer Motion for all animations

Hard constraints:

- Do NOT use Next.js.
- Do NOT create any backend, API routes, or server code.
- Do NOT add authentication of any kind.
- Do NOT use Firebase, Supabase, or any database.
- All data must come from local mock data files.
- Cart state lives in React state (Context + useReducer or a custom hook). No persistence needed.

## 2. Design Direction — This Is the Most Important Section

The site must feel like a top-tier performance brand — think **Gymshark, Ghost, Optimum Nutrition, Alphalete**. Bold, energetic, dark, and athletic — but still clean and premium. It must NOT look like a generic e-commerce template.

**Color system:**

- Background: deep charcoal black (`#0B0D0E`) and near-black (`#060708`) — never flat gray panels
- Surface cards: dark graphite (`#131619`) with subtle 1px borders (`rgba(255,255,255,0.07)`)
- Primary text: crisp white (`#F7F8F9`)
- Secondary text: cool gray (`#8A9199`)
- **Primary accent: electric volt green (`#B4FF39`)** — for prices, active states, primary CTA buttons, hover effects, and energy accents
- Secondary accent: electric cyan (`#2DD4FF`) — used sparingly for badges, category tags, and gradient touches
- Occasional subtle green→cyan gradients on CTAs and section glows — energetic but tasteful, never rainbow

**Typography:**

- Headings: a strong condensed/athletic sans (e.g., "Oswald", "Archivo Black", or "Bebas Neue" via Google Fonts), often UPPERCASE with tight tracking for big headlines
- Body/UI: Inter or Manrope
- Small uppercase eyebrow labels with wide tracking above section headings (e.g., `FUEL YOUR GRIND`, `THE STACK`) in volt green
- Headlines are heavy and impactful (700–900 weight) — the opposite energy of a luxury serif site

**Motion & feel (Framer Motion):**

- Punchy but smooth: fade + upward reveal (`y: 30 → 0`) on scroll, staggered children, `duration: 0.4–0.6`, ease `[0.22, 1, 0.36, 1]`
- Product image hover: quick scale (`1 → 1.06` over 400ms) with a subtle volt-green glow shadow appearing under the card
- Buttons: hover lift with a green glow (`box-shadow` with the volt color at low opacity), slight scale on press
- Modal: fade + scale from 0.95 with backdrop blur
- A subtle animated gradient glow (green/cyan, very low opacity) behind the hero headline
- Respect `prefers-reduced-motion`

**Surface details:**

- Rounded corners (`rounded-xl` / `rounded-2xl`), soft dark shadows
- Glassmorphism on the sticky navbar (dark translucent + `backdrop-blur`) and modal backdrop
- Thin volt-green divider lines between major sections
- Small angled/skewed accent shapes or diagonal section dividers are welcome for athletic energy — used sparingly

## 7. Currency — JOD

All prices in **Jordanian Dinar (JOD)** through a single utility used everywhere:

```ts
export const formatPrice = (amount: number) =>
  new Intl.NumberFormat("en-JO", {
    style: "currency",
    currency: "JOD",
    minimumFractionDigits: 0,
    maximumFractionDigits: 0,
  }).format(amount);
```

No hardcoded "$" anywhere. Realistic supplement prices: roughly 15–70 JOD.
